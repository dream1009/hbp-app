### Poptip API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
|------|------|------|------|------|
| validator | 触发方式 | `String` | click | - |
| placement | 位置 | `String` | top | - |
| title | 标题 | `String` | - | - |
| content | 内容 | `String` | - | - |
| width | 宽度 | `String` | - | - |
| confirm | 确实模式 | `Boolean` | false | - |
| okText | 确定 | `String` | - | - |
| cancelText | 取消 | `String` | - | - |
| transfer | - | `Boolean` | - | - |
| popperClass | 样式 | `String` | - | - |
| wordWrap | 文字换行 | `Boolean` | false | - |
| padding | 边距 | `String` | 8px 16px | - |
