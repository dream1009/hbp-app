### PalettaButton API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
|------|------|------|------|------|
| content | 文本 | `String` | `` | - |
| offset | 位置 | `Number` | Math.PI / 4 | - |
| direction | 方向 | `String` | `lt t rt this.radius rb b lb l 取值有8个方向，左上、上、右上、右、右下、下、左下、左，默认为左上lt` | - |
| radius | 弧度 | `Number` | 90 | - |
| mainButtonStyle | 样式 | `String` | - | - |
